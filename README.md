# Leitor de arquivos XML padrão ANS

Trata-se de um software de conversão desenvolvido para processar arquivos no formato XML que seguem o padrão de schema da ANS. A ferramenta realiza a leitura, interpretação e transformação desses dados, convertendo-os em arquivos no formato Excel (.xlsx). Além disso, o sistema organiza e concatena as informações extraídas, distribuindo-as em diferentes planilhas ou arquivos de acordo com o tipo de dado identificado, o que facilita a análise, o tratamento e a utilização posterior das informações.
